This repository contains the raw data files for the Molecular Epidemiology tutorial created by the Bahl Lab for CIERS training in Taiwan. Empty folders that will be used in the tutorial have been added. 

Data was retrieved from GISAID and represents HA protein nucleotide data as well as metadata from Asia in 2017. 

For Mac users the compiled program for RAxML has been provided.
